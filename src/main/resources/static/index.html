About This Project

A full-stack CRUD application built for learning Spring Boot development.

Fast CRUD

Add, edit, delete and view records instantly without page reloads.

Marks Tracker

Track student marks with live grade calculation and progress bars.

Responsive UI

Works seamlessly across mobile, tablet, and desktop screens.

REST API

Connected to Spring Boot RESTful backend running on port 9090.

CSV Export

Export all student records to CSV with one click from the navbar.

JPA Persistence

Data stored and managed via Spring Data JPA with MySQL.
